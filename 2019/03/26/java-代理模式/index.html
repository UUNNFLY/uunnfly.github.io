预备知识： 关联、聚合和组合

  • 关联（association）,仅仅是两个对象有联系，一个对象用到了另一个对象

  • 组合（composition），在一个对象生命周期内创建了另一个对象，当Foo死亡时，Bar也会死亡

  • 聚合（aggregation）,一个对象借到了另一个对象，但是Foo的生命周期结束时，Bar的生命周期不一定结束
